# Deprecated Multi-Agent Workflow

更新时间：2026-04-29

本文档已降级为 historical reference only。

当前仓库不再默认采用多 Dev Agent 分工流程，也不再把 planning / execution / review 拆成固定工作流。后续 Dev Agent 应以根 `AGENTS.md`、`docs/README.md`、`docs/delivery/tasks/_active.md`、当前 task 和 handoff 为准。

保留本文件只为解释历史 task / handoff 中曾出现过的协作语境。不要把这里的旧流程作为当前任务授权、状态升级或完成判断依据。
